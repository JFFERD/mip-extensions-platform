# mip-3490-site

mip-3490-site 二级联动切换插件

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-3490-site/mip-3490-site.js

## 示例

### 基本用法
```html
<mip-3490-site>
   <select id='pvc' name='pvc'></select>
   <select id='city' name='city'></select>
</mip-3490-site>
```
