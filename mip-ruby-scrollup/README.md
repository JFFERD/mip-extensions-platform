# mip-ruby-scrollup

mip-ruby-scrollup 用来支持 MIP 中内容向上滚动展示，支持图片。

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-ruby-scrollup/mip-ruby-scrollup.js
